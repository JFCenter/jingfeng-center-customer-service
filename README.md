### 示例对话

**用户**：消费为什么没积分？
**客服**：您好，消费后积分未到账通常有以下几种原因：

1. 页面提示"商圈配置失败"——请联系人工客服或拨打025-86155999
2. 该店铺不支持积分——请留意店铺门口或收银台标识
3. 未开通景枫中心无感积分授权——小程序→我的→无感积分→授权开通
4. 微信支付积分未到账——小程序→我的→无感积分→快速积分申请→申请积分
5. 非同手机号支付——积分仅统计到支付手机号对应的会员账户

排除以上情况后仍未解决，请联系积分咨询客服为您查询补录~

## 核心功能

### 1. 会员积分服务
- 积分规则：100积分=1元现金抵扣
- 无感积分开通/关闭指引
- 拍照积分操作路径
- 签到打卡连续奖励（最高31天全勤奖）

### 2. 商场导览
- 育婴室、母婴室位置
- 无障碍/亲子卫生间位置
- 寄存柜位置及容量
- 儿童推车租借（1F/B3/B1）

### 3. 停车缴费
- 收费标准：早8点-晚8点 **4元/小时**；晚8点-早8点 **1元/小时**
- 积分抵扣停车费
- 充电桩：负三层F区（特斯拉/极氪/蔚来）

### 4. 商户查询
支持查询313个品牌商户的楼层位置和联系电话，包括：
- 国际奢品：CHANEL、DIOR、YSL、LAMY
- 美妆护肤：科颜氏、雅诗兰黛、植村秀、NARS
- 餐饮美食：CGV影城、蔡澜港式点心、捞王、巴奴
- 汽车品牌：特斯拉、蔚来、理想、小鹏、华为汽车

### 5. 线上商城
- 小程序退款（1~2个工作日到账）
- 秒杀商品核销路径
- 优惠券/停车券查询
